# BMW Showroom Frontend

## Overview

BMW Showroom Frontend is a modern web application developed as part of my Full Stack Development Internship at GetStack Technologies. The project provides an interactive and user-friendly platform for exploring BMW vehicles, managing user preferences, and booking test drives.

## Features

* Animated Landing Page
* BMW Vehicle Catalogue
* Vehicle Search and Filtering
* Detailed Car Information Pages
* Wishlist Management
* Shopping Cart Functionality
* Customer Reviews Section
* User Profile Management
* Test Drive Booking System
* Responsive User Interface

## Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Vite
* React Router DOM
